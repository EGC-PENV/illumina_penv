==========
 rand_pre
==========

lumlp input : Nom du fichier de luminosité sans numéro de zone ni extension. Par exemple, pour "ete_568_lumlp_01.pgm" on écrit "ete_568_lumlp".
reflect input : Nom du fichier de réflectance sans extension. Par exemple "ete_568_reflect".
ox : Position en x du point d'observation.
oy : Position en y du point d'observation.
r1 : Rayon où la fonction de probabilité en 1/r touche 1.
r2 : Rayon où la fonction de probabilité en 1/r touche pr2.
pr2: Probabilité à r2.
rm : Rayon à l'intérieur duquel la probabilité est réajustée à 1.
imp: Importance relative de l'intensité lumineuse sur la fonction de probabilité.

===========
 rand_post
===========

original lumlp file : Nom du fichier de luminosité sans numéro de zone ni extension. Par exemple, pour "ete_568_lumlp_01.pgm" on écrit "ete_568_lumlp".
input file : Nom du fichier à transformer. (sans extension)
